# Store Metadata — Draft Review Checklist

> All copy in this directory is a DRAFT. It must be reviewed before publication.

## Islamic compliance review (mandatory)

- [ ] All descriptions use the correct honorific ﷺ (sallallahu alayhi wa sallam) consistently.
- [ ] No figurative imagery of the Prophet ﷺ or the Companions is referenced or implied.
- [ ] Category names and question references are verified against `CONTENT_VALIDATION.md`.
- [ ] A qualified Islamic scholar or subject-matter expert has reviewed the store copy.

## Store policy review

- [ ] Google Play Education category confirmed appropriate.
- [ ] Age rating: IARC questionnaire completed (expected result: 3+).
- [ ] Privacy policy URL added to Play Console (can state "no data collected").
- [ ] Developer account name and contact email set in Play Console.
- [ ] Changelogs updated for each release (increment the version code in `changelogs/`).

## Copy accuracy

- [ ] `title.txt` matches the app display name exactly.
- [ ] `short_description.txt` is ≤ 80 characters (Google Play limit).
- [ ] `full_description.txt` is ≤ 4000 characters.
- [ ] French translations reviewed by a native French speaker.

## Assets

- [ ] 512×512 icon uploaded to `images/icon/icon.png` (both locales or one shared copy).
- [ ] Feature graphic (1024×500) in `images/featureGraphic/featureGraphic.png`.
- [ ] At least 2 phone screenshots in `images/phoneScreenshots/`.
