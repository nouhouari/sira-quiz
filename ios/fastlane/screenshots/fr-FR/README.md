# Captures d'écran App Store — Français (fr-FR)

Placez les captures d'écran ici sous la forme : `01_<description>.png`, `02_<description>.png`, etc.

## Types d'appareils et dimensions requis

| Classe d'appareil    | Résolution            | Notes                           |
|----------------------|-----------------------|---------------------------------|
| iPhone 6,7" (Pro Max)| 1290×2796 px          | Requis depuis iOS 16            |
| iPhone 6,5" (Plus)   | 1242×2688 px          | Ancienne taille requise         |
| iPad 12,9" (Pro)     | 2048×2732 px          | Requis si application universelle|

- Maximum 10 captures par type d'appareil.
- PNG ou JPEG, espace colorimétrique RVB, sans alpha.
- Les captures sources se trouvent dans `screenshots/` à la racine du dépôt.
